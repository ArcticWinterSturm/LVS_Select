# hedging so a downstream PHOTO EDITOR + SCORER receives only
# discriminative visual facts. Conservative = readable English
# preserved; only redundant / non-visual / wrong text removed or
# collapsed. No abbreviation mangling (see aggressive variant).

# Expanded from dataset2 (~800 captions, folk festivals + fashion).
#
# Three operation modes:
#   phrase alone on line       →  DELETE the phrase
#   verbose phrase | shorter   →  REPLACE with shorter
#   |phrase                    →  DELETE + auto-capitalize next
#
# NEVER STRIPPED: blurry, sharp, high quality, high ISO, shallow
# depth of field, bokeh, lens, exposure, overexposed, underexposed,
# grainy, noise, chromatic aberration, motion blur, long exposure,
# aperture, shutter speed, any genuine technical photographic term.
